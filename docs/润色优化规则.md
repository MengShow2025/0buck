# 0Buck v8.5 内容润色与优化全流程标准 (Refinery SOP)

本协议是 0Buck “真相引擎”与“欲望引擎”的核心执行标准。旨在通过对阿里巴巴国际站（ICBU）原始数据的深度重塑，将其包装为“大牌平替”的科学真相，从而消弭低价疑虑，建立工匠级信任。

---

## 一、 标题命名标准 (Title Protocol)

**核心公式：** 
**`{核心产品词} - {针对性痛点解决方案} | {物理审计参数 1} & {物理审计参数 2} | 0Buck Verified Artisan`**

*   **顶格爆破**：将“产品-痛点药方”置于标题最前端（0偏移量），确保在移动端搜索结果中 100% 优先展示，极大提升点击率（CTR）。
*   **品质封印**：仅在标题末位保留 `0Buck Verified Artisan`，作为物理审计与工匠品质的最后一道“真理印章”。
*   **参数审计**：中间段落必须包含通过 API 507580 抓取的原始规格（如：*140W GaN*, *IPX7*）。
*   **去工厂化**：AI 自动过滤阿里标题中的供应商名称、型号、起订量（MOQ）等 B2B 词汇。

**示例（3C 数码）：**
`Magnetic Power Bank - Stop Overheating While Charging | 20W Fast Charge & 10000mAh | 0Buck Verified Artisan`

## 二、 “欲望引擎”叙事逻辑 (Description Narrative)

采用 **“五段论”** 黄金堆栈，将“便宜”解释为“供应链科学”。

#### **第一段：[The Hook] 痛点打击与共情**
*   **动作**：调用 **Amazon Pain Points API**，提取同款大牌的 Top 差评。
*   **文案**：直接向品牌溢价开火。*“还在为 $159 的品牌 Logo 买单，却在忍受它不到一年的电池寿命吗？”*

#### **第二段：[The Transformation] 理想状态描述**
*   **动作**：描绘拥有产品后的具体生活画面。
*   **文案**：*“想象一下，现在你可以享受 [产品带来的终极便捷]，而无需顾虑昂贵的维护成本。”*

#### **第三段：[The Logic] 品牌税真相解构**
*   **动作**：公开供应链链路，解构价格构成。
*   **文案**：*“这款产品出自全球 [代工领域] 顶尖工厂。我们砍掉了 30% 平台费、20% 广告费和 15% 品牌税。你支付的每一分钱，都只为物理材料本身买单。”*

#### **第四段：[The Fulfillment Truth Table] 履约真相表 (强制)**
*   **动作**：在描述顶部插入标准 HTML 表格。
*   **字段**：
    *   **Origin (发货地)**：真实海外仓位（如：*US East Warehouse*）。
    *   **Speed (时效)**：实时 DDP 预测时效（如：*3-7 Days*）。
    *   **Protocol (认证)**：标注 `Verified by Alibaba Official API 507580`。

#### **第五段：[The Evidence] 物理审计日志**
*   **动作**：1:1 还原阿里 API 抓取的原始规格（材质、尺寸、电压、克重）。
*   **文案**：*“无修饰参数 (Raw Specs)——我们对每一项物理指标负责。”*

## 三、 视觉资产处理标准 (Visual Assets)

*   **1:1 全量同步**：阿里国际站有多少张详情图，必须 1:1 全部同步，禁止人为删减。
*   **无痕净化准则 (No-Smudge Policy)**：
    *   **去中文化**：严禁出现中文字符。利用 AI 自动重绘背景，彻底移除水印、电话。
    *   **去品牌化**：利用 AI 移除原始工厂 Logo，保留产品本身的工业质感。
*   **视频卡片**：NORMAL 和 REBATE 级别商品必须挂载阿里原厂的 16:9 生产/演示视频。

## 四、 自动化处理流程 (The Workflow)

1.  **数据抓取 (Capture)**：通过 API 507580 抓取阿里原始标题、属性、全量图片和视频。
2.  **痛点对撞 (Collision)**：AI 搜索 Amazon 同款 ASIN，提取 Pain Points 数据。
3.  **灵魂润色 (Soul Polish)**：由 AI 执行“欲望引擎”脚本，将“阿里真相”与“Amazon 痛点”进行对撞叙事。
4.  **注入与隔离 (Inject & Fence)**：
    *   自动注入 **Truth Table**。
    *   根据 `user_country` 挂载地域标签（如 `LOC-US`），执行“本地仓硬隔离”。
5.  **比价更新**：自动在描述中显著展示 `60% OFF AMAZON PRICE` 的对比水印。

---

## 五、 审查红线 (Review Redlines)

1.  **禁令**：图片像素级内容严禁出现中文，严禁出现马赛克。
2.  **物性硬锁**：叙事卖点严禁虚构，必须基于物理审计结果。
3.  **时效隔离**：详情页展示的时效必须与用户的 `user_country` 严格对应。
4.  **定价死守**：MAGNET 必须标记为 $0.00；NORMAL/REBATE 售价必须为 Amazon 售价的 60%。
