# 模块一：VCC 交互架构 - 手工验收清单

> 在浏览器打开 http://localhost:5173，按以下步骤逐项验证

## 1.1 全局框架与响应式

- [ ] **开屏与遮罩**
  - 打开页面，SplashScreen 动画是否正常展现？
  - 遮罩层是否严格限制在手机框内（不溢出到大屏幕）？

- [ ] **主题切换**
  - 点击右上角主题切换按钮（暗色/亮色）
  - 背景、文字颜色、阴影是否平滑过渡？

- [ ] **多语言切换**
  - 点击语言切换（中文/English）
  - UI 界面词条是否实时更新？

- [ ] **大屏适配**
  - 在桌面端（≥1024px），界面是否正确居中呈现"手机框"？
  - 是否带有发光网格背景？
  - 窗口缩放时是否会发生顶部裁切？

- [ ] **品牌 Logo**
  - 顶部 Header 是否使用了统一的 `ob-wordmark` 样式？
  - 是否有异常拉伸？

## 1.2 沙龙与通讯录 (Lounge & Contacts)

- [ ] **双 Tab 切换**
  - 沙龙顶部的"聊天 (Chat)"和"发现 (Discover)"Tab 切换是否顺畅？
  - 下划线动画是否跟随？

- [ ] **+ 号菜单**
  - 点击右上角 + 号，下拉菜单排序是否为"1. 添加好友、2. 发起群聊、3. 扫一扫"？

- [ ] **通讯录管理**
  - 联系人列表（ContactsDrawer）能否正常打开？

- [ ] **好友申请**
  - "新的朋友（NewFriendsDrawer）"抽屉能否打开？
  - 同意/忽略操作后，状态是否持久化存入 `localStorage`，刷新不丢失？

- [ ] **黑名单联动**
  - 在联系人列表中点击"拉黑"，好友是否立刻从列表消失并进入"黑名单（BlacklistDrawer）"？
  - 在黑名单中"解除拉黑"，好友是否能立刻恢复？

## 1.3 核心聊天与 AI 管家 (ChatRoom)

- [ ] **群聊菜单**
  - 在群聊中点击右上角 `...`，是否弹出"成员管理"、"退群"、"关群"菜单？

- [ ] **退群/关群隐藏**
  - 点击"退群"后，该群是否立即从沙龙会话列表中消失？

- [ ] **隐藏群恢复**
  - 沙龙列表底部是否出现"恢复隐藏群"按钮？
  - 点击后群组是否重新出现？

- [ ] **AI 系统指令**
  - 输入特定语言（如"打开分享"），AI 是否能下发系统动作（System Action）如 `OPEN_SHARE_MENU`，并成功唤起对应的抽屉？

## 1.4 推广分享与裂变 (ShareDrawer)

- [ ] **抽屉唤起**
  - 点击分享按钮能否正常滑出 ShareDrawer？

- [ ] **奖励面板展示**
  - 顶部是否正确拉取并显示了当前用户的身份、Tier、分销比例和粉丝奖励比例？

- [ ] **悬停动态提示**
  - 鼠标 Hover 到卡片类型（如 Invite Card）或平台图标（如 WhatsApp）上时，底部的蓝色说明区是否会动态切换对应的文案？

- [ ] **分享生成**
  - 选择平台并点击生成卡片，是否能模拟生成分享链接/模板？
