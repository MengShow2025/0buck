3、将单品去阿里国际 对比相似度（除品牌外）标题、描述、图片 大于 80% 的商品
4、再通过价格比对。亚马逊销售价 *60% （我们销售价）/ CJ 拿货价（仅商品价格） 大于等于 1.5 X  小于 4X 的选入 促销商品/普通商品 。 大于等于 4X 选入 返现商品   。亚马逊划线价（如果没有划线价，即同步使用真实销售价）。新增 0 元购商品，当亚马逊运费大于 阿里国际拿货总价（商品+运费） 时，平台可以推出付运费 0 元购。（即用户仅支付运费可以买走商品）
5、运费和税费，用户需额外支付（即我们不包邮）根据用户的收货地址 CJ 自动计算出的我们的拿货运费，用户需支付。税费根据用户收货地区的税务政策（CJ 向我们拿货收，我们就向用户收）。
