		签到页
			订单 签到返现功能
				用户购物，并确认收货后可以获得签到返现
从确认收货 555天内
分 20期签到返现，每期返还订单扣除运费、税费 后实付金额的 5%
第一期：连续签到 5天 获得 5%
第二期：连续签到 10天获得 5%
第3-20期 连续签到 30天获得 5%
签到奖励规则
1、第一期还没签到完成前（即确认收货后 5天内）可以选择分享 3 人购物免单（分享免单不能同签到奖励同时享受）
2、开启第一期签到奖励发放后不能再选择分享免单
3、每期必须连续签到才能获得返现
4、任何一期签到中断，即失去本期签到返现的奖励。并下次签到开启下一期的第一天计算
5、如果 555天时间到了，已连续签到的天数奖励计算（奖励=天数/305% 奖励基数）
6、根据规则，用户最多完全签到将获得 20期返现，也就是最大返现总额=奖励基数
			某一期断签及本期奖励作废，即下一次签到从下期第 1 日开始算
				每期最多可以使用一次续期卡
				续签卡获得：积分兑换
				每期仅可使用一次，限制最大断签时间不能大于 5 天。可使用并延续本期
			签到返现规则说明入口

			各种奖励规则说明入口
				用户拼团免单规则
					 1、用户购买商品后，从对应订单分享入口分享给好友购买相同订单
					 2、并用户在自己订单还没第一期签到完成时自动选择拼团免单奖励
						如果已经分享有 3 人以上购买同商品订单的情况下
						用户也可以选择签到返现，但不能两个奖励重叠
					 3、商品订单分享链接 3 人以上购买并确认收货后奖励=全返对应自己商品订单的金额
					 4、如果购买多件同商品即奖励=商品件数 * 3 人 以上，即每 3人返还一件金额
				用户粉丝奖励规则
					自己专属邀请码
					邀请新用户注册
					新用户注册后 2年内，所有购物都可以获得奖励，奖励=除去运费和税费实际支付金额 * （1.5-3%）根据不同的级别
					对方用户确认收货后发放到系统钱包，自行提现
				达人分销规则
					不用区分是否是新会员
					商品专属分销链接，确认收货及奖励（扣除运费+税费的所有实际支付金额）
					奖励=基数 *  8%-20% （根据审核和谈判）一次性奖励
					用户有效确认收货且过退货期，平台收到款项后发放到平台对应钱包余额中，自行提现
				达人粉丝奖励规则
					专属链接邀请的新会员
					 2年内被邀请的新注册用户的所有订单都可以获得奖励（每次订单一次性奖励）
					奖励比例=3%-8%
					用户有效确认收货，平台收到款项后发放到平台对应钱包余额中，自行提现
				所有奖励发放：订单有效确认收货且过退货期，平台收到款项后发放到平台对应钱包余额中，自行提现
				（奖励优先级：分销 > 拼团免单>粉丝奖励）。及当某订单发送重叠冲突时

后端
	AI
		主AI 
			模型：Gemini 2.5 / Claude 4.5
			解析理解分拆用户需求
			提供人性化解答和规划
			制定最适合的推荐计划，和商品
		搜索查询 agent
			模型：DeepSeek v3 / 本地模型
			根据主AI 的规划，从数据库优化后的数据内搜索提取对应的商品和商家
		客服/查询 agent
			模型：Gemini 2.5
			完成用户对活动、规则、订单、物流、客服咨询、投诉等的知识库内规则回答和展示
		寻货 agent
			模型：Gemini2.5
			定时搜索热门商品和类目，寻找货源
			对比找到的拿货货源和相应的亚马逊上的销售价格比对，符合利润率要求的推送给铺货工具
			外部AI 自动谈判供应商
		优化Agent 
			模型：Gemini2.5
			优化数据库：商品、商家、活动、的内容和翻译
			根据主AI 分析的用户画像和商品智能标签，对比和写入数据库
		热门和经常调用的可以采用后台记录以后直接调用的方式节约 token 费用
