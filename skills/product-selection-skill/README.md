# 选品分析 Skill

**版本：** 1.0.0 | **分类：** 跨境电商 / 选品决策

## 概述

对跨境电商候选产品进行七维度系统评估，输出结构化选品报告与入场策略建议。适合铺货前决策、品类扩张评估、多产品优先级排序。

## 核心能力

- 七维度评估：需求度 / 竞争强度 / 利润潜力 / 运营难度 / 差异化空间 / 合规风险 / 物流友好度
- 加权综合评分（满分10分）
- 利润快速估算（含 FBA 费用、广告费、退款损耗）
- 多产品横向对比
- 入场策略与备货建议
