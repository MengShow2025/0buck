# 选品合规风险速查

## 一、高风险品类（谨慎选择）

### 🔴 极高风险（新手不建议）

| 品类 | 风险原因 |
|------|---------|
| 锂电池独立产品 | 航空危险品，Amazon 严格管控，易被下架 |
| 无人机 | FAA 注册要求，出口管制，频率认证复杂 |
| 激光笔（Class 3B/4） | FDA 管制，多国禁售 |
| 电子烟/雾化器 | Amazon 明确禁售，各国严格监管 |
| 仿制品/高仿 | 商标侵权，直接封号 |
| 成人用品 | 平台限制，海关敏感 |
| 武器/弹药配件 | 进出口严格管制 |
| 药品/处方产品 | 需医疗资质，无资质违法 |

### 🟡 中等风险（需要认证，可做）

| 品类 | 需要的认证 | 获取难度 |
|------|-----------|---------|
| 电器产品（美国） | FCC / UL | 中等，需实验室测试 |
| 电器产品（欧盟） | CE（LVD + EMC + RED） | 中等 |
| 电器产品（日本） | PSE | 中等 |
| 电器产品（韩国） | KC | 中等 |
| 儿童玩具（美国） | CPSC / ASTM F963 | 中等 |
| 儿童玩具（欧盟） | CE / EN71 | 中等 |
| 膳食补充剂 | FDA DSHEA 备案 | 高 |
| 医疗器械（一类） | FDA 510(k) | 极高 |
| 有机认证产品 | USDA Organic | 高 |
| 食品接触材料 | FDA 食品级证明 | 中等 |

### 🟢 低风险（适合新手）

```
手机壳/保护套
数据线（非认证声称类）
手机支架
收纳盒/整理箱
厨房手动工具（非电动）
文具/办公用品
书包/收纳袋
装饰性家居（非电器）
宠物玩具（非食品）
运动健身小器械（非电子类）
```

## 二、专利风险高发品类

以下品类外观专利/实用专利纠纷频发，选品时需重点排查：

### 极高专利风险
```
磁吸充电器（MagSafe 相关）
无线充电板（部分结构专利）
蓝牙音箱（外观专利多）
折叠手机支架（机构专利）
指尖陀螺（2016–2018年专利战）
磁力积木（多家专利纠纷）
平衡车（Segway 专利）
```

### 专利排查方法

**免费工具：**
- Google Patents：patents.google.com
- USPTO（美国专利局）：patents.uspto.gov
- 搜索关键词：产品英文名 + 「design patent」/「utility patent」

**排查步骤：**
```
1. Google Patents 搜索产品英文名
2. 筛选「Active」（有效）专利
3. 查看申请日期（2000年前的专利通常已过期）
4. 对比专利图与你的产品外观，判断相似度
5. 有疑问可咨询 Derechos IP 等跨境专利服务机构
```

## 三、Amazon 平台限制品类

以下品类在 Amazon 上架需要额外审批：

### 需要申请才能销售（Gated Category）
```
美妆/个护（某些子类）
食品/饮料
健康/家庭护理
珠宝
手表
```

### 申请所需材料（一般）
```
- 品牌授权书（如销售品牌产品）
- 供应商发票（3个月内）
- 产品合格证书
- 部分需要实地核查
```

## 四、海关敏感品类

以下产品在中国出口或目标国进口时可能遭遇海关问题：

| 品类 | 敏感原因 | 注意事项 |
|------|---------|---------|
| 对讲机/无线电设备 | 需无线电型号核准 | 确认是否有频率认证 |
| 夜视仪 | 军民两用，出口管制 | 需出口许可证 |
| 望远镜（专业军用级） | 同上 | 慎选 |
| 隐藏摄像头 | 部分国家禁止销售 | 需标注摄像功能 |
| 仿制枪（玩具枪外观逼真） | 多国有颜色/外观要求 | 需符合橙色枪口标准 |
| 种子/植物 | 检疫限制 | 禁止跨境邮寄（大部分国家）|
| 酒精饮料 | 进口许可/税率复杂 | 不适合跨境电商 |

## 五、快速合规评估问卷

选品时逐一回答以下问题，「是」越多风险越高：

```
□ 产品含有锂电池？（独立电池产品更敏感）
□ 产品是电器？（需认证）
□ 产品面向 14 岁以下儿童？（安全认证）
□ 产品涉及食品/入口/皮肤接触？（FDA/食品级）
□ 产品有无线通信功能？（FCC/CE RED）
□ 产品有拍摄/录音功能？（隐私合规）
□ 产品仿制了知名品牌外观？（商标/外观专利）
□ 产品在淘宝/1688 上能搜到大量超低价同款？（专利被控风险）
□ 产品是否有医疗/保健功效声称？（FDA 管制）
```

**0–2 个「是」：** 合规风险低，可优先考虑
**3–5 个「是」：** 需专项核查认证要求
**6+ 个「是」：** 高风险，建议更换选品方向
