# 竞争格局评估指南

## 一、竞争强度核心判断指标

### 指标 1：头部卖家 Review 数量

Review 数量是新卖家入场门槛的最直观指标：

| 头部10名平均 Review 数 | 竞争评级 | 新品机会 |
|----------------------|---------|---------|
| < 100 | 极低竞争 | 🌟 绝佳机会 |
| 100–500 | 低竞争 | ✅ 好机会 |
| 500–1,000 | 中等竞争 | ⚠️ 需差异化 |
| 1,000–3,000 | 高竞争 | ⚠️ 门槛高 |
| > 3,000 | 极高竞争 | ❌ 新品极难 |

### 指标 2：头部卖家集中度

分析搜索结果前 10 名的卖家分布：

**低集中度（分散竞争，好机会）：**
- 10个卖家来自 10 个不同品牌
- 没有任何卖家占据超过 3 个 listing
- 没有 Amazon 自营（AmazonBasics）

**高集中度（垄断市场，难进入）：**
- 1–2 个品牌占据 5+ 个排名
- Amazon 自营出现在前 3
- 知名大品牌（Anker/Belkin/YETI 等）占主导

### 指标 3：Best Seller / Amazon's Choice 分布

```
Amazon's Choice：算法推荐，表示该关键词下最具竞争力的产品
→ 若 Amazon's Choice 是新品（< 1年）：说明市场仍在更新，有机会
→ 若 Amazon's Choice 是老品（>3年）且 Review > 5,000：市场固化，难突破
```

### 指标 4：价格带分析

绘制前 20 名的价格分布：
```
示例：耳机类目价格分布
$0–$15:  ████ 4个（低端红海）
$15–$30: ██████████ 10个（主战场，最激烈）
$30–$50: ████ 4个（中端，有差异化空间）
$50–$80: ██ 2个（高端，溢价机会）
$80+:    0个（空白区间？）
```

**策略建议：**
- 避开产品数量最密集的价格段
- 寻找产品数少但销量尚可的价格区间
- 空白价格带未必是机会，先验证需求

## 二、竞品质量分析（Review 挖掘）

### 差评分析（寻找改进机会）

通过竞品 1–2 星 Review 找改进方向：

**操作方法：**
```
1. 打开竞品 listing
2. 筛选 1 星和 2 星 Review
3. 找出重复出现的抱怨点（出现 3+ 次即为系统性问题）
4. 记录：质量问题 / 功能缺陷 / 包装问题 / 尺寸问题 / 说明书问题
```

**高价值差评信号：**
```
「电池寿命太短」→ 可以做更大电池版本
「尺寸偏小/偏大」→ 提供多尺寸选项
「包装太简陋」→ 升级包装作为差异化
「说明书看不懂」→ 提供多语言+视频说明
「做工粗糙」→ 用材升级作为卖点
「颜色与图片不符」→ 提升主图真实性
```

### 好评分析（确认核心卖点）

通过 4–5 星 Review 确认买家最看重什么：
```
高频出现的正面词 = 维持并在标题/五点描述中强调
```

## 三、竞争强度评分模型

基于以下因素综合评分（满分 10 分，分越高 = 竞争越弱 = 越值得进入）：

| 评估项 | 权重 | 10分标准 | 5分标准 | 1分标准 |
|-------|------|---------|---------|---------|
| 头部平均 Review 数 | 30% | < 100 | 300–500 | > 3,000 |
| 头部卖家集中度 | 25% | 高度分散 | 中等集中 | 高度垄断 |
| 是否有亚马逊自营 | 20% | 无 | 不在前5 | 排名前3 |
| 价格带空白机会 | 15% | 明显空白 | 小空白 | 无空白 |
| 新品上升势头 | 10% | 多个新品进前10 | 少数新品 | 无新品 |

## 四、竞争类型识别与应对策略

### 类型 A：高需求 + 低竞争（蓝海）
**特征：** 搜索量大，但头部 Review 少，卖家分散
**策略：** 快速入场，占位，加速 Review 积累
**代表：** 新兴品类早期，如早期的智能家居配件

### 类型 B：高需求 + 高竞争（红海）
**特征：** 搜索量大，头部品牌强，Review 数千
**策略：** 细分切入（特定人群/场景/价格段），差异化产品
**代表：** 手机壳、无线耳机主流价位段

### 类型 C：低需求 + 低竞争（蓝海小市场）
**特征：** 搜索量小，竞争也弱，但市场规模有限
**策略：** 低库存试水，适合多 SKU 铺货策略
**代表：** 特定小众配件、细分工具

### 类型 D：低需求 + 高竞争（最差）
**特征：** 搜索量小，但已有强势品牌占据
**策略：** 放弃或大幅调整产品方向
**代表：** 已被品牌私域化的小众品类

## 五、竞争分析 SOP

```
Step 1：搜索核心关键词，截图前 20 名
Step 2：统计头部 10 名的 Review 数量（计算平均值）
Step 3：识别品牌集中度（几个品牌占据前10）
Step 4：确认是否有亚马逊自营
Step 5：标注价格带分布，寻找空白区间
Step 6：各取 1–2 名竞品，读 20 条差评，记录痛点
Step 7：综合评分，输出竞争强度得分
```
