# 市场需求信号识别指南

## 一、免费需求验证方法

### 1. Amazon 搜索框自动补全

在 Amazon 搜索框输入产品词，观察：
- 联想词数量（越多说明搜索量越大）
- 是否出现「best xxx」「xxx for xxx」等长尾词
- 记录前 5 个联想词作为核心关键词

**操作步骤：**
```
1. 打开 amazon.com（不登录，避免个性化影响）
2. 输入产品词，截图记录联想词
3. 换用英文近义词重复操作
4. 统计高频出现的修饰词（即买家真实需求点）
```

### 2. Amazon 类目 BSR 榜单分析

通过 BSR（Best Seller Rank）判断市场规模：

**估算月销量公式（经验值）：**
| BSR 排名（Electronics） | 估算月销量 |
|------------------------|-----------|
| 1–100 | 3,000–50,000+ |
| 101–500 | 1,000–3,000 |
| 501–2,000 | 300–1,000 |
| 2,001–5,000 | 100–300 |
| 5,001–10,000 | 50–100 |
| > 10,000 | < 50 |

> 注：不同品类换算系数不同，服装/家居 BSR 与销量关系与电子类有差异

**操作步骤：**
```
1. Amazon 搜索产品词
2. 点击前 5-10 名产品，查看页面底部「Product information」中的 BSR
3. 结合上表估算月销量
4. 前 10 名月销量之和 = 该关键词大致市场规模
```

### 3. Google Trends 趋势判断

验证需求是上升、稳定还是衰退：

```
网址：trends.google.com
操作：
1. 搜索产品英文名
2. 时间范围选「过去 5 年」，判断长期趋势
3. 时间范围选「过去 12 个月」，判断季节性
4. 地区选「United States」
```

**趋势解读：**
- 上升趋势：新兴品类，早期进入有先发优势
- 稳定趋势：成熟市场，竞争激烈但需求确定
- 下降趋势：衰退品类，慎重进入
- 季节性波动：注意备货时机，Q4 旺季前 2–3 个月备货

### 4. eBay 已售数量

查看竞品在 eBay 上的已售数量，验证真实购买行为：
```
搜索产品词 → 勾选「Sold listings」→ 查看价格与销量
```

## 二、付费工具验证（更精准）

### Helium 10（推荐）

| 功能 | 用途 |
|------|------|
| Cerebro | 竞品 ASIN 反查关键词搜索量 |
| Magnet | 关键词搜索量与趋势查询 |
| Black Box | 按条件筛选产品（搜索量/BSR/Review 数） |
| Xray | 实时查看产品月销量和收入 |

**选品核心指标：**
```
搜索量：主关键词月搜索量 > 5,000（太小市场太窄）
竞争度：搜索量/竞争商品数 > 5（机会指数）
月销量：目标排名位置月销量 > 300 件
Review 数：头部产品 Review < 500（新品有机会）
```

### 卖家精灵 / 蜘蛛精（中文工具）

适合中国卖家，数据涵盖 Amazon 各站点：
- 产品机会发现器：按维度筛选低竞争蓝海
- 关键词工具：中文界面，操作简便
- 竞品监控：跟踪竞品销量变化

## 三、需求信号评分标准

### 高需求信号（+2分每项）
```
✅ Google Trends 5年呈上升趋势
✅ 主关键词月搜索量 > 50,000
✅ BSR 前 10 平均月销量 > 1,000 件
✅ 搜索联想词丰富（>10个）
✅ Amazon Choice 标签存在（平台认可需求）
```

### 中等需求信号（+1分每项）
```
🟡 Google Trends 趋势稳定（无明显上升或下降）
🟡 主关键词月搜索量 10,000–50,000
🟡 BSR 前 10 平均月销量 300–1,000 件
🟡 搜索联想词中等（5–10个）
```

### 低需求信号（-1分每项）
```
❌ Google Trends 呈下降趋势
❌ 主关键词月搜索量 < 10,000
❌ BSR 前 10 平均月销量 < 300 件
❌ 高度季节性（旺季/淡季相差 5 倍以上）
❌ 搜索联想词少（< 5个）
```

## 四、常青品类 vs 季节性品类对照

### 常青品类（全年稳定，推荐新手）

**电子类：**
手机壳、数据线、充电器、蓝牙耳机、电脑配件、摄像头配件

**家居类：**
收纳整理、厨房用品、床品、清洁工具、灯具

**宠物类：**
宠物食碗、玩具、牵引绳、梳毛工具

**个护类：**
护发工具、美甲工具、按摩器

### 季节性品类（需把握入场时机）

| 品类 | 旺季 | 建议入场时间 |
|------|------|-----------|
| 圣诞装饰 | 11–12月 | 9月前备货 |
| 泳池/水上玩具 | 6–8月 | 4月前备货 |
| 万圣节 | 10月 | 8月前备货 |
| 返校用品 | 8–9月 | 6月前备货 |
| 园艺工具 | 3–6月 | 1月前备货 |
| 暖手宝/电热毯 | 10–2月 | 8月前备货 |

## 五、需求验证 SOP 流程

```
Step 1: Google Trends 趋势判断（5分钟）
        → 判断是否值得继续调研

Step 2: Amazon 搜索框联想词采集（10分钟）
        → 确认核心关键词

Step 3: BSR 月销量估算（15分钟）
        → 评估市场规模

Step 4: 若工具可用，Helium 10 Xray 验证（10分钟）
        → 精确月销量和收入数据

Step 5: 需求评分汇总
        → 输出需求维度得分
```
