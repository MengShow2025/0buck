## 二、全链路利润测算公式

```
【收入端】
毛收入 = 售价（Selling Price）

【成本端】
总采购成本（USD）= (采购单价 + 包装 + 质检) ÷ 汇率
头程运费（USD）  = 运费总额 ÷ 汇率 ÷ 批次件数
FBA 配送费（USD）= 按尺寸档查表
平台佣金（USD）  = 售价 × 佣金率（Amazon 通常 8–15%）
广告费（USD）    = 售价 × ACOS × 广告占比
仓储费（USD）    = 月仓储费 × 平均库龄（月）÷ 件数
退款损耗（USD）  = 售价 × 退款率 × 损耗系数

【利润】
毛利润 = 毛收入 - 总采购成本 - 头程运费 - FBA 配送费
净利润 = 毛利润 - 平台佣金 - 广告费 - 仓储费 - 退款损耗
净利润率 = 净利润 ÷ 售价
```

## 四、利润风险因子

以下情况会显著压缩利润，评分时需酌情扣分：

### 高风险因子（每项 -1分）

```
退货率高的品类：
- 服装/鞋类：15–30% 退货率
- 电子产品（复杂）：8–15% 退货率
- 尺寸型产品（床品/家具）：10–20%

广告竞争激烈：
- ACOS > 30%（意味着广告费侵蚀大量利润）
- 主关键词 CPC > $2（每次点击成本高）

仓储周转慢：
- 产品单价低 + 体积大（每件仓储成本占比高）
- 季节性产品（淡季大量库存积压）

汇率风险：
- 人民币升值 5% = 利润直接减少（以成本比 40% 算约影响 2 个百分点）
```

### 低风险加分项（每项 +0.5分）

```
高复购率：宠物食品/耗材/个护 → 复购降低获客成本
私标壁垒：已注册品牌，竞品抄袭成本高
捆绑销售：套装价值感强，竞争对比困难
专利保护：外观专利已申请，降低被跟卖风险
```

## 五、利润敏感性分析

不同变量对净利润率的影响（以售价 $30、净利润率 25% 的产品为基准）：

| 变量变化 | 净利润率影响 |
|---------|-----------|
| 采购成本降低 10% | +约 3–4% |
| 广告 ACOS 从 25% → 20% | +约 2% |
| 退款率从 5% → 10% | -约 2–3% |
| 汇率从 7.2 → 6.8 | -约 2% |
| FBA 费涨 $0.5 | -约 1.7% |
| 售价降低 10% | -约 8–10%（杠杆效应显著）|

**核心结论：售价和广告费率是影响利润最大的两个杠杆，优先优化。**
