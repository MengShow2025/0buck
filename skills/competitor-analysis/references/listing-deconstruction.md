# Listing 结构拆解指南

## 一、标题拆解框架

### 标题信息密度分析

将竞品标题拆分为以下元素，统计覆盖情况：

```
品牌词：___________（是否有品牌？是否知名？）
产品核心词：______（核心品类词，搜索量最大的词）
功能/技术词：_____（ANC / Wireless / Bluetooth 5.3 等）
规格/参数词：_____（40H / 2000mAh / 6-in-1 等）
场景/受众词：_____（for Travel / for Office / for Kids 等）
附加价值词：_____（Foldable / Lightweight / Premium 等）
```

**覆盖率评估：**
```
覆盖 5–6 项：信息密度高，SEO 强
覆盖 3–4 项：中等，有优化空间
覆盖 1–2 项：弱，关键词覆盖不足
```

### 标题位置重要性

```
前 80 字符（移动端可见范围）：
← 最关键区域，主关键词必须在这里 →

80–200 字符（桌面端可见范围）：
← 次要关键词、规格补充、场景词 →

超过 200 字符（不推荐）：
← 会被截断，不计入关键词权重 →
```

## 二、五点描述拆解框架

### 叙事结构分析

每个 Bullet Point 通常遵循以下结构之一：

**类型 A：功能型（最常见）**
```
[全大写卖点关键词] — [功能描述]，[使用场景或用户利益]。
例：ACTIVE NOISE CANCELLING — Advanced ANC technology blocks up to 95% of ambient noise...
```

**类型 B：场景型**
```
[全大写场景词] — [用户情景描述]，[产品如何解决这个情景]。
例：PERFECT FOR TRAVEL — Whether you're on a 12-hour flight or a daily commute...
```

**类型 C：数据型**
```
[全大写数字卖点] — [量化描述]，[对比或解释]。
例：50-HOUR BATTERY LIFE — Up to 50 hours of continuous playback with ANC on...
```

### 五点卖点覆盖分析

好的五点描述应覆盖以下 5 个维度：
```
□ 核心功能（最强技术/功能卖点）
□ 使用体验（舒适/便捷/直觉）
□ 兼容性/通用性（多设备/多场景）
□ 耐久性/品质（材质/认证/保障）
□ 额外价值（配件/售后/特殊设计）
```

**分析记录：**
```

卖点1：[关键词] → 属于[类型] → 覆盖[哪个维度]
卖点2：
卖点3：
卖点4：
卖点5：

未覆盖的维度（我们的机会）：_________________
```

## 三、图片策略拆解

### 标准图片序列分析

Amazon 最多 9 张图片，竞品通常的排列逻辑：

| 位置 | 常见内容 | 优秀竞品的做法 |
|------|---------|-------------|
| 图1（主图） | 白底产品图 | 白底+最佳角度，清晰展示轮廓 |
| 图2 | 卖点信息图 | 最强卖点+数据，如「40H Battery」 |
| 图3 | 场景使用图 | 目标用户使用场景，情感共鸣 |
| 图4 | 多角度/细节图 | 做工细节，材质质感 |
| 图5 | 功能拆解图 | 多功能标注，技术特性展示 |
| 图6 | 对比图 | 与竞品/上代产品对比（合规范围内）|
| 图7 | 尺寸/规格图 | 精确尺寸，消除购买疑虑 |
| 图8 | 配件清单图 | 包装内容物展示 |
| 图9 | 品牌故事/认证图 | 品牌背书，信任感建立 |

### 图片分析记录表

## 五、关键词策略推断

### 从 Listing 反推关键词矩阵

通过分析竞品 listing 的词汇，推断其关键词布局：

```
步骤1：提取标题中所有名词/形容词/动词
步骤2：提取五点描述中的高频技术词汇
步骤3：提取产品描述中的场景词和受众词
步骤4：组合推断可能的 Search Terms
```

**示例（ANC 耳机）：**
```
标题词汇：wireless, headphones, bluetooth, noise cancelling, foldable, travel
五点词汇：ANC, battery, comfortable, lightweight, multipoint
场景词汇：commute, office, studying, zoom calls, gaming

推断 Search Terms 可能包含：
- wireless headphones for commuting
- foldable noise cancelling headphones
- bluetooth headphones zoom calls
- lightweight anc headset office
```

## 六、竞品运营策略识别

### 广告策略痕迹

```
Sponsored 标签：
- 搜索结果中出现 Sponsored 标注 = 在投 SP 广告
- 在竞品 Listing 页出现其他产品广告 = 该竞品被竞争对手投放 SD 广告

「Frequently bought together」：
- 可推断竞品的捆绑销售策略
- 可发现关联品类的机会

「Customers also viewed」：
- 反映真实的竞争关系
- 找到隐性竞争对手
```

### 促销策略识别

```
□ 是否有 Coupon（优惠券百分比）
□ 是否有 Lightning Deal 痕迹（BSR 短期异常飙升）
□ 是否有 Prime Exclusive Discount
□ 是否参与 Subscribe & Save（耗材类）
□ 变体是否有「Save X% when buying together」
```

### 品牌建设程度

```
评估指标：
□ 是否有 Amazon Store（品牌店铺）
□ 是否完成 Brand Registry
□ 是否有 A+ Content
□ 是否有品牌 Story
□ 是否有多语言 listing（欧洲站）
□ 社交媒体是否活跃（外部流量）
```
