# 竞品分析 Skill

**版本：** 1.0.0 | **分类：** 跨境电商 / 竞品研究

## 概述

对跨境电商竞品进行八维度全面拆解，覆盖卖点策略、定价逻辑、评价洞察、视觉策略、关键词布局、运营数据，输出结构化竞品分析报告及差异化切入建议。

## 核心能力

- 八维度系统拆解：定位/定价/评价/视觉/关键词/运营/弱点/差异化
- Review 深度挖掘（5星好评验证 + 1-2星痛点提取 + 3星期望落差分析）
- Listing 结构拆解与关键词反推
- 差异化策略生成（产品层/Listing层/运营层）
- 多竞品横向对比矩阵
