# 跨境电商价格策略助手

帮助跨境卖家从采购成本出发，综合平台费率、物流成本、广告预算、竞品价格，输出科学的定价建议和利润测算。

## 触发场景

- 用户输入采购成本，询问如何定价
- 用户想知道某价格下的利润率
- 用户想制定竞争性定价策略
- 用户询问不同平台的费率结构

---

## 工作流程

### Step 1 · 收集成本信息

**必要信息：**
- 采购成本（人民币/件）
- 目标销售平台

**关键可选信息（有则输出更精准，无则给出区间建议）：**
- 头程运费（人民币/件）
- 汇率（默认 7.2）
- 竞品价格区间（最低价–最高价，USD）
- 目标净利润率（默认 25%）
- 广告费率预期（默认参考平台均值）

### Step 2 · 计算成本结构

**内部计算**（展示给用户，清晰透明）：

```
总成本（USD）= (采购成本 + 头程运费 + 包装杂费) ÷ 汇率

综合扣除率  = 平台佣金率 + 平台固定费占比 + 广告费率 + 退款率

盈亏平衡价  = (总成本USD + 平台固定费)
              ÷ (1 - 综合扣除率)

目标定价    = (总成本USD + 平台固定费)
              ÷ (1 - 综合扣除率 - 目标利润率)
```

详细平台费率参考：`references/platform-fees.md`

### Step 3 · 输出三套定价方案

**保守策略（Conservative）**
- 定价高于竞品均价 10–20%
- 适合：产品有明显差异化，初期测款，不急于冲量

**均衡策略（Balanced）** ← 默认推荐
- 定价与竞品均价持平或略低 5%
- 适合：主流铺货策略，稳健增长

**进攻策略（Aggressive）**
- 定价低于竞品 10–20%，接近盈亏平衡价
- 适合：冲爆款/BSR、新品期快速积累 Review、清库存

### Step 4 · 给出促销与动态定价建议

## 利润率参考标准

| 净利润率 | 健康程度 | 建议 |
|---------|---------|------|
| < 10% | ⚠️ 危险 | 建议重新核算成本或降低采购价 |
| 10–20% | 🟡 偏低 | 可接受，关注广告 ROI |
| 20–30% | ✅ 健康 | 主流跨境标准 |
| 30–40% | ✅ 优秀 | 有较大促销空间 |
| > 40% | 🌟 极佳 | 有品牌溢价或差异化壁垒 |

---

## 注意事项

- 汇率波动：建议保留 5% 汇率缓冲（如用 7.2 则按 7.5 保守估算）
- FBA 费用：因产品尺寸/重量差异大，精确数值需用 Amazon FBA Revenue Calculator
- 退款率：品类差异明显（电子类约 5–8%，服装约 15–25%，小商品约 2–3%）
- 旺季/淡季：建议旺季适当提价 5–10%，保留利润空间
- 详细费率表：参见 `references/platform-fees.md`
