# 定价策略场景示例集

## 示例 1：无线蓝牙耳机（Amazon 美国）

**输入信息：**
- 采购成本：¥45 / 件
- 头程运费：¥18 / 件
- 汇率：7.2
- 竞品区间：$19.99 – $49.99
- 目标利润率：25%

**成本测算：**
```
总成本（USD）= (45 + 18) ÷ 7.2 = $8.75
平台佣金（15%）：按定价估算
FBA 配送费：约 $4.20（小件标准）
广告费（12%）：按定价估算
退款预留（5%）：按定价估算
综合扣除率：≈ 32%（含广告+退款）

盈亏平衡价 = ($8.75 + $4.20) ÷ (1 - 0.32) = $19.04
目标定价 = ($8.75 + $4.20) ÷ (1 - 0.32 - 0.25) = $30.12
```

**三套方案：**
| 策略 | 定价 | 净利率 | 每件净利 |
|------|------|-------|---------|
| 保守 | $39.99 | 33% | $13.20 |
| 均衡 ⭐ | $29.99 | 24% | $7.20 |
| 进攻 | $22.99 | 9% | $2.07 |

```

```

## 示例 3：LED 氛围灯（Shopee 泰国）

**输入信息：**
- 采购成本：¥12 / 件
- 头程运费：¥8 / 件
- 汇率：7.2
- 竞品区间：THB 199 – 599（约 $5.70 – $17.10）
- 目标利润率：20%

**成本测算：**
```
总成本（USD）= (12 + 8) ÷ 7.2 = $2.78
Shopee 泰国佣金（2%）：按定价估算
交易费（2%）：按定价估算
广告费（7%）：按定价估算

盈亏平衡价 = $2.78 ÷ (1 - 0.11) = $3.12 ≈ THB 109
目标定价 = $2.78 ÷ (1 - 0.11 - 0.20) = $4.04 ≈ THB 141
```

## 常见定价错误

### ❌ 错误 1：忽略广告成本
- 很多卖家只算佣金和运费，漏掉广告费
- Amazon 新品期 ACOS 常高达 40–60%
- **修正**：广告费单独列项，按保守估算预留

### ❌ 错误 2：用人民币利润率衡量
- 直接用（售价-成本）÷ 售价 ≠ 实际利润率
- 忽略了汇率损耗（收款通道通常折扣 1–3%）
- **修正**：全程用 USD 计算，最后换算 CNY 收益

### ❌ 错误 3：没有考虑退款成本
- 服装类退款率可达 20–30%
- 每次退款不仅损失商品，还要承担 FBA 退货处理费
- **修正**：按品类设置退款预留比例

### ❌ 错误 4：旺季不提价
- Q4（10–12月）Amazon 仓储费翻 3 倍
- 竞争激烈期广告费上涨 30–50%
- **修正**：Q4 提价 10–15%，保护利润空间

### ❌ 错误 5：定价低于成本只算眼前
- 进攻定价应有明确的时限（3个月内冲到 BSR XX 名）
- 没有计划的低价策略会耗尽现金流
- **修正**：制定退出低价策略的触发条件

## 动态定价策略

### 新品期（前 30 天）
- 采用「均衡」或「进攻」定价，加速积累 Review
- 配合 5–15% Coupon 提升转化率
- 广告预算加大，建立 BSR 排名

### 成熟期（有稳定 Review 后）
- 逐步提升至「均衡」或「保守」价格
- 每次提价幅度 ≤ 10%，观察转化率变化
- 维持 Coupon 吸引价格敏感用户

### 旺季（Q4、618、双11）
- 提前 2–4 周恢复正常/提高价格
- 旺季大促前设置 Coupon，让买家感知折扣
- 旺季后恢复，避免利润侵蚀

### 竞品跟价策略
- 竞品降价时：不必跟降，先观察 3–5 天看转化影响
- 若自身转化率下降 >15%，可跟降 5%
- 避免「价格战」陷阱，关注利润绝对值

## 汇率风险管理

| 汇率变动 | 对利润影响（以总成本$10、售价$30为例） |
|---------|--------------------------------------|
| 7.2（基准） | 净利润基准 |
| 7.5（+4%） | 成本降 3.8%，净利润+约8% |
| 6.8（-6%） | 成本升 5.9%，净利润-约13% |

**建议：**
- 定价时汇率保守使用（如实际7.2，按7.0计算）
- 使用 Wise / PingPong / 连连支付 等收款，减少汇损
- 大量采购/库存时考虑适当对冲
