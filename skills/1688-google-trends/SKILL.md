# 1688-alphashop-google-trends

1688遨虾AI Google Trends 趋势分析 Skill - 查询关键词搜索趋势，辅助选品决策。

## 概述

本 skill 调用 1688遨虾AI选品系统的 Google Trends 趋势分析API（HSF接口），根据用户提供的关键词和目标区域，查询 Google Trends 上的搜索热度数据，帮助判断：

- **市场需求趋势**：关键词搜索量的增长或下降趋势
- **季节性特征**：识别搜索热度的峰值月份，判断最佳销售时机
- **关键词对比**：同时查询多个关键词，对比市场热度差异
- **新兴趋势**：发现快速增长的品类和关键词

## 核心功能

### 支持的查询能力
- ✅ **批量查询**：最多同时查询5个关键词
- ✅ **全球市场**：支持美国、英国、日本、德国等多个国家/地区
- ✅ **历史数据**：返回近12个月的搜索热度变化
- ✅ **峰值识别**：自动识别搜索热度最高的月份
- ✅ **趋势可视化**：时间序列数据便于分析趋势

### 返回数据
每个关键词返回以下信息：

#### 基础信息
- **关键词**：查询的关键词
- **趋势数据**：近12个月的搜索热度时间序列（0-100标准化分数）
- **峰值月份**：搜索热度最高的月份（格式：yyyy-MM）

## 数据解读指南

### 趋势数据说明
- **数值范围**：0-100
- **100 分**：该关键词在近12个月内的搜索热度峰值
- **其他分值**：相对于峰值的百分比
- **0 分**：搜索量极低或无数据

### 趋势类型判断

#### 📈 上升趋势（增长市场）
```
趋势数据：30 → 40 → 55 → 70 → 85 → 100
```
- **特征**：搜索热度持续增长
- **意义**：新兴品类或增长市场
- **建议**：适合入局，抢占先机

#### 📉 下降趋势（衰退市场）
```
趋势数据：100 → 85 → 70 → 55 → 40 → 30
```
- **特征**：搜索热度持续下降
- **意义**：衰退品类或饱和市场
- **建议**：谨慎入局，或寻找差异化

#### 🔄 周期性波动（季节性商品）
```
趋势数据：20 → 30 → 50 → 80 → 100 → 60 → 30 → 25
```
- **特征**：明显的峰谷交替
- **意义**：季节性商品
- **建议**：提前备货，抓住旺季

#### ➡️ 平稳趋势（常青品类）
```
趋势数据：45 → 48 → 50 → 52 → 48 → 50
```
- **特征**：搜索热度稳定
- **意义**：常青品类，需求稳定
- **建议**：适合长期经营

## 最佳实践

### ✅ 推荐做法

1. **使用目标市场的语言**
   - 英语市场用英文关键词
   - 日本市场用日语关键词
   - 避免语言不匹配导致数据偏差

2. **选择具体、有代表性的关键词**
   - ✅ 好："wireless earbuds"
   - ❌ 差："earbuds"（过于宽泛）
   - ❌ 差："best wireless earbuds under 50"（过于具体）

3. **同时查询核心词和长尾词**
   ```bash
   python3 scripts/google_trends.py query \
     --keywords "bluetooth headphones" "wireless bluetooth headphones" \
     --region US
   ```

4. **结合其他数据综合判断**
   - Google Trends 反映搜索需求
   - 需结合销量、价格、评分等维度
   - 不要仅依赖趋势数据做决策

---

### ❌ 避免的做法

1. **过于宽泛的关键词**
   - ❌ "phone"（太宽泛，数据噪音大）
   - ✅ "iPhone 15 Pro Max"（具体明确）

2. **查询过多关键词**
   - ❌ 一次查询10个关键词（超过限制）
   - ✅ 最多5个关键词

3. **忽略区域差异**
   - ❌ 用英文关键词查询日本市场
   - ✅ 用日语关键词查询日本市场
