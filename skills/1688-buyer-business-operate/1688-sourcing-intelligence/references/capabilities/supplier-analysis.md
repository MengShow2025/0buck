# 供应商分析指南

## CLI 调用

```
python3 {baseDir}/cli.py supplier --query "关键词" [--method 方式] [--limit 数量]
```

| 参数 | 默认 | 说明 |
|------|------|------|
| --query | 必填 | 搜索关键词/URL/图片URL |
| --method | keyword | 搜索方式：keyword/url/image |
| --limit | 10 | 返回数量，最大20 |

## 供应商评估维度

| 维度 | 关注点 | 优秀标准 |
|------|-------|---------|
| 经营年限 | 是否可靠 | ≥3年 |
| 好评率 | 服务质量 | ≥90% |
| 响应时间 | 配合度 | <2小时 |
| 资质认证 | 合规性 | 有必要认证 |
| 复购率 | 长期价值 | ≥15% |

## 供应商对比规则

### 推荐标准（满足3项及以上）

1. 经营年限 ≥ 3年
2. 好评率 ≥ 90%
3. 响应时间 < 2小时
4. 有必要资质认证
5. 复购率 ≥ 15%

### 风险标记

- 经营年限 < 2年（经验不足）
- 好评率 < 85%（服务风险）
- 响应时间 > 24小时（配合度差）
- 无资质认证且品类需要（合规风险）

## 展示规范

Agent 回复必须包含：
1. 供应商列表（原样输出 markdown）
2. 综合评估（各维度分析）
3. 对比建议（多供应商时）
4. 下一步行动（询盘/查看详情）
