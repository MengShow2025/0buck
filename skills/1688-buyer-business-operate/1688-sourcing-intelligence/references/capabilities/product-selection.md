# 选品指南

## CLI 调用

```
python3 {baseDir}/cli.py search --query "商品描述" [--channel 渠道] [--limit 数量]
```

| 参数 | 默认 | 说明 |
|------|------|------|
| --query | 必填 | 自然语言描述，API自行理解语义 |
| --channel | "" | 下游渠道：douyin/taobao/pinduoduo/xiaohongshu |
| --limit | 20 | 返回数量，最大50 |

## 选品决策规则

### 推荐标准（满足3项及以上）

1. 30天销量 ≥ 500
2. 好评率 ≥ 90%
3. 复购率 ≥ 10%
4. 下游铺货数 < 300（蓝海）
5. 24h揽收率 ≥ 90%

### 风险标记（命中任一项须说明）

- 好评率 < 85%（售后风险）
- 下游铺货数 > 500（红海竞争）
- 标题含品牌名（侵权风险）
- 单价极低(<5元)且非日用品（质量隐患）

## 展示规范

Agent 回复必须包含：
1. 商品列表（原样输出 markdown）
2. 选品分析（基于 data 分析）
3. 风险提示（如有）
4. 下一步引导

## 异常处理

| 场景 | 表现 | Agent 应对 |
|------|------|-----------|
| 搜索无结果 | product_count: 0 | 建议调整关键词重试 |
| 商品缺少数据 | stats 为 null | 注明"缺少数据指标" |
